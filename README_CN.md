
# Code Generator


`CodeGenerator`，是用于`Intellj IDEA`的代码生成器插件，可以根据数据库实体对象(Entity/Domain)，通过自定义或是使用默认的模板，生成对应的`Java`代码。

## 插件特性

- 支持自定义模板
- 支持模板导入导出
- 支持生成 Restful 风格的API

## 配置模板

此插件支持多模板（多模板组）， 你可以使用预定义的变量自行编辑模板；编辑完成之后可以通过右下角 `Validate` 按钮验证模板是否正确。
