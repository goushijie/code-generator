You can use the following predefined variables in templates
  • ${clazz} the selected class
    • ${name} the class name
    • ${packageName} the class name
    • ${fields} the class fields
      • ${name} the field name
      • ${type} the field type
  • ${fn} String tools
    • ${pluralize}
    • ${singularize}
    • ${decapitalize}
    • ${capitalize}
    • ${dcp} decapitalize and pluralize
  • ${BASE_PACKAGE} user selected base package
  • ${USER} current user system login name
  • ${YEAR} current year
  • ${MONTH} current month
  • ${DAY} current day
  • ${DATE} current system date
  • ${TIME} current system time
  • ${DATE_TIME} current system dateTime
